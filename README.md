# Manny D' Great - Developer Dashboard Portfolio

A premium, GitHub-style developer portfolio dashboard built with Next.js 15, TypeScript, TailwindCSS, and Framer Motion.

## Features
- **GitHub Aesthetics**: Dark mode, grid layout, and contribution heatmap.
- **Glassmorphism UI**: Premium visual effects with backdrop blurs and subtle glows.
- **Live Data**: Integated with GitHub REST API for profile, repositories, and activity timeline.
- **Interactive Components**: 
  - Dynamic Contribution Graph
  - Animated Skill Matrix
  - Terminal-style Contact Section
  - Hover-elevated Project Cards
- **Responsive Design**: Optimized for both desktop and mobile views.

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Styling**: TailwindCSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Language**: TypeScript

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

Deploy easily to Vercel:
```bash
npx vercel
```
